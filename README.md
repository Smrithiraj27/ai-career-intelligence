# 🚀 AI Career Intelligence System

An AI-powered web application that analyzes resumes, predicts suitable career roles, and provides personalized career guidance using NLP and Machine Learning.

---

## 🔥 Features

- 📊 ATS Score Calculation (Weighted Scoring System)
- 🧠 NLP-based Resume Matching (TF-IDF + Cosine Similarity)
- 🎯 Career Role Prediction (AI-based logic)
- 📉 Skill Gap Analysis (Job + Role based)
- 🤖 AI Feedback (Smart insights like ChatGPT)
- 🛤️ Personalized Career Roadmap
- 📄 Resume Parsing using PyMuPDF

---

## 🧠 Tech Stack

- **Frontend:** HTML, CSS, Bootstrap
- **Backend:** Flask (Python)
- **AI/NLP:** Scikit-learn (TF-IDF, Cosine Similarity)
- **Libraries:** PyMuPDF, Regex

---

## ⚙️ How It Works

1. Upload your resume (PDF)
2. Enter a job description
3. System extracts resume content
4. Performs:
   - Skill detection
   - NLP similarity analysis
   - Career role prediction
5. Outputs:
   - ATS score
   - Missing skills
   - AI feedback
   - Career roadmap

---

## 📊 Example Output

- ATS Score: 78%
- Role: Data Scientist
- Missing Skills: Deep Learning, Statistics
- Feedback: Resume moderately aligned
- Roadmap: Step-by-step learning plan
